⚠  FIREBASE EI OLE SEADISTATUD  ⚠

Leidsime, et firebase config on endiselt vaikeväärtustega.
Palun järgi allpool olevaid juhiseid:

1. Mine aadressile: https://console.firebase.google.com
2. Loo uus projekt (nt "kernel-chat")
3. Lisa veebirakendus ( ikoon)
4. Kopeeri oma firebaseConfig objekt
5. Ava see index.html fail tekstiredaktoris
6. Otsi üles sektsioon "FIREBASE KONFIGURATSIOON" ja asenda vaikeväärtused

const firebaseConfig = {
  apiKey: "SINU_API_KEY_SIIA",
  authDomain: "SINU_PROJECT.firebaseapp.com",
  databaseURL: "https://SINU_PROJECT-default-rtdb.firebasedatabase.app",
  ...
};

NB! Realtime Database peab olema lubatud Firebase konsoolis:
Build → Realtime Database → Create Database → Start in test mode
