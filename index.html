Leidsime, et firebase config on endiselt vaikevГӨГӨrtustega.
Palun jГӨrgi allpool olevaid juhiseid:

1. Mine aadressile: https://console.firebase.google.com
2. Loo uus projekt (nt "kernel-chat")
3. Lisa veebirakendus ( ikoon)
4. Kopeeri oma firebaseConfig objekt
5. Ava see index.html fail tekstiredaktoris
6. Otsi Гјles sektsioon "FIREBASE KONFIGURATSIOON" ja asenda vaikevГӨГӨrtused

PГӨrast seadistamist lae leht uuesti.

// Koht koodis, kus muudatused teha (rea nr ~320):

  apiKey: "SINU_API_KEY_SIIA",
  authDomain: "SINU_PROJECT.firebaseapp.com",
  databaseURL: "https://SINU_PROJECT-default-rtdb.firebaseio.com",
  projectId: "SINU_PROJECT_ID",
  storageBucket: "SINU_PROJECT.appspot.com",
  messagingSenderId: "SINU_SENDER_ID",
  appId: "SINU_APP_ID"
};
